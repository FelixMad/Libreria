Iconos

fg-icon para incoportar el bloque icono.

El bloque "Icono" viene con una serie de caracteristicas de base. Estas se pueden variar incoporando modificadores preestablecidos.

Modificador - Iconos con distintas grafias

fg-icon_graph_*ICONO* es el modificador del tipo de icono.

Modificador - Iconos con colores

fg-icon_color_*COLOR* es el modificador del color del icono.

Modificador - Iconos en tamaños

fg-icon_size_*TAMAÑO* es el modificador del tamaño del icono.

Ejemplo de iconos

class="fg-btn fg-icon_graph_*ICONO* fg-icon_color_*COLOR* fg-icon_size_*TAMAÑO*"

Botones

fg-btn para incoportar el bloque "Boton" y tiene las siguientes caracteristicas de Elementos y Modificadores:

  • Los Elementos que pueden ser añadidos son __icon, __text y __badge. Dependiendo de lo que se quiera visualizar.
  • fg-btn viene con una serie de caracteristicas de base. Estas se pueden variar incoporando Modificadores preestablecidos _color_*COLOR* y _size_*TAMAÑO.

Elementos - Botones con icono y texto

Este botón (class="fg-btn") se se puede componer internamente con elementos preestablecidos:

  • "Icono" con class="fg-icon fg-btn__icon fg-btn__icon_float_left".
  • "Texto" con class="fg-text fg-btn__text".
